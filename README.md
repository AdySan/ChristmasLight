# ChristmasLight
Relay controlled by webserver running on an ESP8266

## Features

 - Uses SPIFFS filesystem to store webpages
 - Supports file uploads to SPIFFS filesystem via browser
 - Support OTA firmware update (using ArduinoOTA)
